# Skill Generator

Use this skill whenever you need to create a new skill for the VUELTA F1NAL.

## Instructions
1. **Location**: Create in `.agent/skills/<name>/SKILL.md`.
2. **Metadata**: Include `name` and `description` in frontmatter.
3. **Scope**: Focus on UI/UX, Component design, and Kotlin-specific features.
